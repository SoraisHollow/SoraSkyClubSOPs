1. Updating the Notion ‚ÄúEvents‚Äù Database

Goal: Keep the internal SoraSkyClub event board up to date so staff can see who is doing what and when.

1.1 Create or update the Notion event

1.2 Internal notes (optional)

  • Links to playlists, sponsor info, or world-dev notes.

2. Creating the VRChat Group Scheduled Event

Goal: Make sure the event is visible to the VRChat community and the SoraSkyClub group.

2.1 Create the event

  1. Go to the VRChat website and open the SoraSkyClub group.
  2. Navigate to Events and select Create Scheduled Event.
  3. Fill out the form:
    • Title: Match or closely mirror the Notion description.
    • Date & Time: Use the ‚ÄúWhen?‚Äù you finalized in #üóíÔ∏è„Éªplanning.
    • World: Choose the correct SoraSkyClub world or event world.
    • Description: Brief, clear, and on-brand. Include who the event is for and any requirements.
  4. Save / publish the event.

2.2 Record the VRChat link

Copy the event link and paste it back into:

  • The Notion event page (link section or notes).
  • The event message in Discord (for easy joining).

3. Creating the Discord Event

Goal: Give the community an easy in-Discord way to see, RSVP, and share the event.

3.1 Set up the event

  1. In the main SoraSkyClub Discord server, open the appropriate channel or the server header menu.
  2. Click Create Event.
  3. Choose the correct type (e.g. ‚ÄúExternal‚Äù if it links out to VRChat).
  4. Fill in:
    • Event Name: Match the VRChat event title.
    • Start Time: Same as the VRChat event.
    • Description: Short description + mention the VRChat group event.
    • Location / Link: Use the VRChat event link you copied.
  5. Upload an on-brand banner image if available.
  6. Publish the event.

3.2 Announce in planning & event channels

  • Post the Discord event link in #üóíÔ∏è„Éªplanning so staff can confirm.
  • Share it in the public event/announcement channel so community members can RSVP.

4. Creating the Fourthwall Post (Website Event Post)

  4. In the body, include:

    • Where (VRChat group & world name) + link to VRChat event.
    • Link to the Discord event, if applicable.
  5. Visibility: Set the post to Public so it shows on the main website.
  6. Publish and double-check that it appears on the site.
